[digits]

Syntax

[digits: format; digits]

Description

Specifies the current digit formatting mode for numbers.

Parameters

  • mode format

    The digit format to use.

    • normal: Use as many digits as necessary to accomodate the number.
    • pad: Pad numbers to a specific number of digits.
    • truncate: Truncate numbers over a specific number of digits.
  • number digits

    The digit count to associate with the mode.
