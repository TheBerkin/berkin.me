[chance]

The chance function is a block attribute that specifies the probability that the next block will be executed.

Syntax

[chance: chance]

Description

Modifies the likelihood that the next block will execute. Specified in percentage.

Parameters

  • number chance

    The percent probability that the next block will execute.

Examples

Random exclusion

The following example demonstrates the behavior of a chance call over several iterations.

[rs:20;\n] { Always prints. [chance:50]{\sSometimes prints.} }

Generally, about half of the lines written by this pattern will contain the sentence affected by chance.
