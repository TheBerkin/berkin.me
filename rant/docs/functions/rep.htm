[rep]

Aliases: r

The rep function is a block attribute that is used to modify the repetition count of the next block.

Syntax

[rep: times]

Description

Sets the repetition count for the next block.

Parameters

  • number times

    The number of times to repeat the next block.

Examples

Repeating a letter

The following example demonstrates simple usage of rep on static text, in this case, a letter.

[rep:10]{X}

The code inside the block is repeated ten times, and as a result, the text is printed ten times.

XXXXXXXXXX

Multiple items in a repeater

Repeaters also work with multi-item blocks, as seen in the following example.

[rep:16]{A|B|C|D|E|F|G} EFGGBFDDGABEEGCD
