[numfmt]

The numfmt function controls how printed numbers are formatted. This does not affect static text, unless it's inside a num call.

Syntax

[numfmt: format]

Description

Sets the current number formatting mode.

Parameters

  • mode format

    The number format to use.

    • normal: No special formatting.
    • group: Group digits with the system's digit separator.
    • group-commas: Group digits by commas.
    • group-dots: Group digits by dots.
    • roman: Uppercase Roman numerals.
    • roman-upper: Uppercase Roman numerals.
    • roman-lower: Lowercase Roman numerals.
    • verbal-en: US English spellings of numbers. Only works with integers.
    • hex: Uppercase hexadecimal.
    • hex-upper: Uppercase hexadecimal.
    • hex-lower: Lowercase hexadecimal.
    • binary: Robot language.
[numfmt: format; rangeAction]

Description

Runs the specified pattern under a specific number formatting mode.

Parameters

  • mode format

    The number format to use.

    • normal: No special formatting.
    • group: Group digits with the system's digit separator.
    • group-commas: Group digits by commas.
    • group-dots: Group digits by dots.
    • roman: Uppercase Roman numerals.
    • roman-upper: Uppercase Roman numerals.
    • roman-lower: Lowercase Roman numerals.
    • verbal-en: US English spellings of numbers. Only works with integers.
    • hex: Uppercase hexadecimal.
    • hex-upper: Uppercase hexadecimal.
    • hex-lower: Lowercase hexadecimal.
    • binary: Robot language.
  • pattern rangeAction

    The pattern to run.

Examples

Roman numerals

This example prints the numbers 1 to 10 in Roman numerals.

[rs:10;\n][numfmt:roman]{[repnum]} I II III IV V VI VII VIII IX X

Verbal numbers

The verbal-en mode will spell out numbers in US English, as demonstrated here:

[numfmt: verbal-en] [num: 2000] two thousand

Binary

If for some reason you need numbers in binary format, Rant has you covered.

[numfmt: binary] [digits: pad; 2] [rs:4;\n] { [rs:4;\s] { [num:0;255] } } 01001001 00110111 11100010 01100111 00000110 10100101 11100001 11111010 01001101 00011110 00111001 10110010 00111101 01010100 11011010 00000111
