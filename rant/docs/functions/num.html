[num]

Aliases: n

The num function generates a random number and prints it in the current number format (specified using numfmt).

Syntax

[num: min; max]

Description

Prints a random number between the specified minimum and maximum bounds.

Parameters

  • number min

    The minimum value of the number to generate.

  • number max

    The maximum value of the number to generate.

[num: input]

Description

Formats an input string using the current number format settings and prints the result.

Parameters

  • string input

    The string to convert into a number.

Examples

The example below shows a simple pattern that generates a list of ten random numbers between 1 and 100.

[rs:10;,\s]{[num:1;100]} 21, 23, 24, 85, 75, 98, 9, 35, 78, 13
