[sync]

Aliases: x

Syntax

[sync: name; type]

Description

Creates and applies a synchronizer with the specified name and type.

Parameters

  • string name

    The name of the synchronizer.

  • mode type

    The synchronization type to use.

    • none: Random ordering.
    • ordered: Execute from left to right.
    • reverse: Execute from right to left.
    • deck: Shuffle items and execute in order. Re-shuffled each time all items are iterated.
    • cdeck: Shuffle items and execute in order. Same order is used for each traversal.
    • locked: Same item is selected each time.
